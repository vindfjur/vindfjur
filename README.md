### 🧰 Featured projects

**🐍 [ephemdir](https://github.com/vindfjur/ephemdir)** — temporary directories that clean
themselves up, after a lifetime or on restart. Cross-platform, on
[PyPI](https://pypi.org/project/ephemdir/).

```bash
pip install ephemdir                 # any platform
brew install vindfjur/tap/ephemdir   # macOS / Linux
```

**🏛️ [Владимир‑33](https://github.com/vindfjur/vladimir33)** — a travel guide to the
Vladimir region of Russia: an interactive map of 2300+ OpenStreetMap places, tourist
routes, architectural heritage and folk‑ornament article pages. Built with **Astro**,
**TypeScript** and **MapLibre GL**, shipped as static, MIT‑licensed.
Created Nov–Dec 2024, rebuilt from scratch in June 2026.

[![Open site](https://img.shields.io/badge/🌐_Open_site-vladimir33.vercel.app-C9A227?style=flat)](https://vladimir33.vercel.app)
