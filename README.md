### 🧰 Featured project

**[ephemdir](https://github.com/vindfjur/ephemdir)** — temporary directories that clean
themselves up, after a lifetime or on restart. Cross-platform, on
[PyPI](https://pypi.org/project/ephemdir/).

```bash
pip install ephemdir                 # any platform
brew install vindfjur/tap/ephemdir   # macOS / Linux
```
